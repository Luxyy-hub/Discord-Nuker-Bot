# Discord-Nuker-Bot
A Discord bot server nuker with voice chat spam

## How to use?

- install [NodeJS](https://nodejs.org/en/download/)

- click on install.bat

- create a bot and paste the token inside the "" in the first string

- then click on nuke.bat
